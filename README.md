*# AI-Driven Virtual Interview Simulator with NLP Assessment*



*An AI-powered interview platform that evaluates candidates using speech recognition, facial emotion detection, NLP-based answer assessment, and automated feedback generation.*



*## Features*



*- Facial Emotion Detection*

*- Speech-to-Text Conversion*

*- NLP Answer Evaluation*

*- Automated Scoring*

*- Gemini AI Feedback*

*- GUI-based Interview Experience*



*## Installation*



*pip install -r requirements.txt*



*## Run*



*python GUI8.py*
